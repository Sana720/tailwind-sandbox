# Privacy Policy for Tailwind CSS Sandbox Max

**Last Updated: June 22, 2026**

Tailwind CSS Sandbox Max ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy details how we handle user data and information when you use our Google Chrome extension.

## 1. Information We Collect
Tailwind CSS Sandbox Max **does not collect, store, track, transmit, or share** any personal information, browsing history, or user data. 

## 2. Permissions Used
To function properly, the extension requests the following permissions from Google Chrome:
- **`activeTab`**: Used solely to display the visual sandbox editor on the webpage currently open and active when you click the extension's toolbar icon. This permission is temporary and sandboxed to the active page.
- **`scripting`**: Used to dynamically execute the inspector highlighting and controls stylesheet on the active webpage.

## 3. Data Processing and Storage
All operations are executed completely **on your local machine** within your browser tab's active session. 
- No webpage content, HTML, or Tailwind utility class configurations are stored locally or sent to external servers.
- The extension does not use any cookies, tracking pixels, or external analytics integrations.

## 4. External Assets
The extension dynamically loads the official Tailwind CSS Play CDN (`https://cdn.tailwindcss.com`) to compile CSS utility classes locally on standard web pages. We do not transmit user or page data to this endpoint.

## 5. Changes to This Privacy Policy
We may update our Privacy Policy from time to time. Any changes will be posted directly within the extension package and updated here.

## 6. Contact Us
If you have any questions or feedback regarding this policy, please reach out to us via our GitHub repository.
